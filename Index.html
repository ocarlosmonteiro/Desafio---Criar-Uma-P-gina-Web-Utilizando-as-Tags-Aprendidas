O que é HTML?

(voltar)

HTML é uma tecnologia utilizada no desenvolvimento de páginas web junto ao CSS que iremos abordar em um outro momento. Formada por elementos para se criar as marcações no corpo das páginas web. A tecnolgia é uma linguagem de marcação e não de programação como alguns pensam e nos tópicos a seguir iremos abordar alguns de seus elementos que são utilizados nas sintaxes básicas de desenvolvimento utilizando a tecnoligia.

O que são Elementos?

Os elementos são "códigos" que são utilizados para gerar as marcações nas páginas web, suas sintaxes são simples, o elemento é formado por um < no inicio, seguido do elemento que deseja utilizar, EX: h1 e em seguida fechado com >.

Exemplo: < h1 >

Tags de titulos ou cabeçalho

(voltar)

Os elementos HTML < h1 > – < h6 > representam seis níveis de título de seção. < h1 > é o nível da seção mais alto e < h6 > é o mais baixo.

Elementos < i >, < strong >, < mark >, < small >, < sup > e < sub >

(voltar)

Italico < i >

O elemento < i > é utilizado para transforar o texto em italico como no exemplo abaixo:

Texto italico: Lorem ipsum dolor sit amet consectetur adipisicing elit. Earum recusandae doloribus ipsum quo cumque dolorum iusto nam dignissimos. Quibusdam fuga earum reiciendis totam cupiditate eos mollitia, consequatur omnis architecto dicta?

Negrito < strong >

O elemento < strong > é utilizado para transformar o texto em italico como o exemplo abaixo:

Texto negrito: Lorem ipsum dolor, sit amet consectetur adipisicing elit. Recusandae saepe tempore quia ducimus, excepturi minus incidunt obcaecati neque eum omnis repudiandae nulla quas impedit, facilis perferendis laudantium tempora ipsum autem!

Marcador < mark >

O elemento < mark > é utilizado para destacar algo no texto, similar a um marca texto.

Texto marcador: Lorem ipsum dolor sit amet, consectetur adipisicing elit. Illo accusantium earum nisi laboriosam exercitationem reprehenderit consequuntur ipsam! Dolorum eius, consectetur perspiciatis excepturi officiis magni inventore, fugit minima, quisquam id quia?

Small < small >

O elemento < small > é utilizado para diminuir o texto, tornado a fonte um pouco menor.

Texto com elemento small: Lorem ipsum dolor sit amet consectetur adipisicing elit. Sapiente ea esse quo repellendus velit magnam. Incidunt itaque libero odit est ullam, asperiores sapiente reiciendis iure deleniti. Nostrum accusamus rem architecto!
