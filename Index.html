O que é HTML?

(voltar)

HTML é uma tecnologia utilizada no desenvolvimento de páginas web junto ao CSS que iremos abordar em um outro momento. Formada por elementos para se criar as marcações no corpo das páginas web. A tecnolgia é uma linguagem de marcação e não de programação como alguns pensam e nos tópicos a seguir iremos abordar alguns de seus elementos que são utilizados nas sintaxes básicas de desenvolvimento utilizando a tecnoligia.

O que são Elementos?

Os elementos são "códigos" que são utilizados para gerar as marcações nas páginas web, suas sintaxes são simples, o elemento é formado por um < no inicio, seguido do elemento que deseja utilizar, EX: h1 e em seguida fechado com >.

Exemplo: < h1 >

Tags de titulos ou cabeçalho

Os elementos HTML < h1 > – < h6 > representam seis níveis de título de seção. < h1 > é o nível da seção mais alto e < h6 > é o mais baixo.
